    Símbolos e álgebra básica¶

    from sympy import Symbol

x = Symbol('x')
y = Symbol('y')

print(x + y + x -y)

    2*x

    a = (x+y)**2
print(a)
print(a.expand())
print(a.subs(x, 1).expand())
print(a.subs(x, 1).expand().subs(y, 1))

    (x + y)**2
x**2 + 2*x*y + y**2
y**2 + 2*y + 1

    Limites¶

    from sympy import Symbol, limit, diff, integrate, sin, oo

x = Symbol('x')
y = Symbol('y')

print(limit(sin(x)/x, x, 0))
print(limit(x, x, oo))
print(limit(1/x, x, oo))

    1
oo
0
